Question 1

box <- c(0, 1, 2, 3, 4, 5)
prior <- rep(1/6, 6)

# E = a single marble is selected and it is green
likelihood <- box/5
product <- prior * likelihood
posterior <- product / sum(product)
bayes_table = data.frame(box,
                         prior,
                         likelihood,
                         product,
                         posterior)
bayes_table |>
  adorn_totals("row") |>
  kbl(digits = 4) |>
  kable_styling()
box prior likelihood product posterior
0 0.1667 0.0 0.0000 0.0000
1 0.1667 0.2 0.0333 0.0667
2 0.1667 0.4 0.0667 0.1333
3 0.1667 0.6 0.1000 0.2000
4 0.1667 0.8 0.1333 0.2667
5 0.1667 1.0 0.1667 0.3333
Total 1.0000 3.0 0.5000 1.0000

I think box 5 is most likely to have been chosen because it has the most green marbles (in fact it only has green marbles).

The posterior probabilities are proportional to the likelihoods since the prior distribution is uniform.
